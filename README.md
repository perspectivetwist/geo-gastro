# GEO Scanner V0

URL eingeben — GEO-Score erhalten. Prüft ob deine Website in KI-Antworten sichtbar ist.

## Stack
Next.js 14, Tailwind CSS, TypeScript, Vercel, Claude Haiku, Jina.ai, Notion API

## Lokale Entwicklung
```bash
npm install
npm run dev
```

## ENV-Variablen
Kopiere `.env.local.example` nach `.env.local` und setze die Keys.
